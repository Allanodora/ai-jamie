# ai-jamie

Voice controlled OBS visual overlay assistant.

## MVP

- Browser overlay for OBS
- Floating visual frame
- Show, hide, next, and back controls
- Add image or video URLs
- Optional voice commands

## Run

```bash
npm install
npm run dev
```

## OBS setup

Add a Browser Source with this URL:

```text
http://127.0.0.1:5173/overlay
```

Use 1080 width and 1920 height for vertical shorts.

## Commands

- Jamie show
- Jamie hide
- Jamie next
- Jamie back
